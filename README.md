# 🍑 うんちくんの大冒険！勉強バトル 💩

小学三年生向けの楽しい勉強ゲームです！

## ✨ 特徴

- **4科目18単元**: 算数、国語、理科、社会の合計1800問
- **楽しいアニメーション**: 正解でキラキラ✨、不正解でうんこ💩
- **スマホ対応**: HTMLで動くので、どのスマホでもプレイ可能
- **中毒性のあるゲームデザイン**: コンボシステムで飽きない
- **音声フィードバック**: 効果音で臨場感UP

## 🎮 使い方

1. `index.html`をブラウザで開く
2. 科目を選ぶ
3. 単元を選ぶ
4. 10問ランダムに出題されます
5. 正解してコンボを重ねて高得点を目指そう！

## 🎵 サウンドについて

ゲームは音声なしでも動作しますが、より楽しむためにWeb Audio APIを使用した簡易的な効果音が鳴ります。
音声のON/OFFは右上のボタンで切り替えられます。

## 📝 問題の追加方法

`questions/`フォルダ内のJSONファイルを編集することで、問題を追加・変更できます。

### JSONファイルの構造
```json
{
  "subject": "科目名",
  "unit": "単元名",
  "level": 3,
  "questions": [
    {
      "id": 1,
      "question": "問題文",
      "choices": ["選択肢1", "選択肢2", "選択肢3", "選択肢4"],
      "correct": 2,  // 正解のインデックス（0始まり）
      "hint": "ヒント（省略可）"
    }
  ]
}
```

## 🌟 遊び方のコツ

- **コンボを重ねよう**: 連続正解すると得点が1.5倍、2倍、3倍...と増えていきます
- **毎日少しずつ**: 1日10問×3単元で楽しく続けられます
- **間違えてもOK**: うんこが出ても大丈夫！何度でもチャレンジしよう

## 💡 保護者の方へ

このゲームは小学三年生の学習内容に沿った問題を収録しています。
- スマホやタブレットで気軽にプレイできます
- オフラインでも動作します
- プレイ時間を決めてご利用ください
- ハイスコアがlocalStorageに保存されます（端末内のみ）
